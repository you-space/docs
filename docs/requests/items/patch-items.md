# PATH /items/:id

Update a item

## Body

### typeId

-   type: `number`

### visibilityId

-   type: `number`

### value

-   type: `object`

### Example

```json
{
    "value": {
        "id": "7e690d9ec04912130daafc55bba7",
        "title": "another title"
    }
}
```

## Response

```json
{
    "message": "item updated"
}
```
