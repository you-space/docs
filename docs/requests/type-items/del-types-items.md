# DEL /types/:type_id/items/:id

Delete a type-item

## Params

### type_id

-   type: `number | string`
-   required: `true`
-   details: id or name of type

### id

-   type: `number`
-   required: `true`
-   details: id of item

## Response

```json
{
    "message": "type-item deleted"
}
```
